# 项目介绍

这是用来打轴的网页。(音频):在线搜索或者本地上传; (歌词):直接复制到歌词框内

[视频教程](https://www.bilibili.com/video/BV1vf4y1c7co/?spm_id_from=333.999.0.0)

## 功能

在线编辑歌词时间轴

在线搜索音频(网易源)

> 注意：由于vip限制有些音频只有30秒左右，需要登录vip账号

编辑历史保存

网易云云盘管理
